# Hi!

### About

Baumhard is an animation library which is focused on animating font glyphs. 

It can be used for any software that wishes to bring font glyphs to life, from simple editors, to UI, 
to games whose graphics are represented partially or totally by glpyhs.

### Features

 - Define simple or complex glyph models in a tree structures
 - Define mutation trees to mutate your models
 - Define animation timelines that are composed of mutation trees
 - Unicode support out of the box
 - Drag and drop font files into your repository and use in autocomplete
 - Or use whatever's on the system
